Question-1: What are the different ways to select an element in the DOM?

Some methods to select an element in the DOM are getElementByID(),getElementsByClassName(),querySelector().

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML sets HTML content of an element, innerText sets visible text content and textContent sets all type of text content without HTML tags.

Question-3: What is event delegation in the DOM?

At event delegation parent element listens for events on its child element using bubbling.

Question-4: What is event bubbling in the DOM?

Event bubbling is a method by which parent elements can handle events for child element without using different event listeners for every child.

Question-5: How do you create, add, and remove elements using JavaScript?

To create elements in Javascript we can use .createElement(), to add elements .appendChild() and lastly to remove element .remove() methods can be used.
